### Browser extensions for MetWord

https://metword.co

遇见单词是一个浏览器划词翻译插件，标记你不认识的单词，在下次遇见时获得提醒。

### Branches

- Branch `main` is of Manifest V2, serving for Firefox and Safari.
- Branch `manifest_v3` is of Manifest V3, serving for Chrome and Edge.

### Build

	$ npm install
	$ npm run watch

### Release

	$ npm install
	$ npm run prod
